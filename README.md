# European Development Indicators - Iceland

## Instructions to run:

0. Make sure you have Python 3.8+ installed and a compatible Jupyter Notebook reader (e.g., VSCode, JupyterLab).

1. Clone the repository to your local machine.

```bash
git clone https://github.com/muffindud/iceland-data-analysis
```

2. Navigate to the project directory.

```bash
cd iceland-data-analysis
```

3. Checkout the desired branch (if applicable).

```bash
git checkout assignment-2
```

4. Create a virtual environment (optional but recommended).

```bash
python -m venv venv
source venv/bin/activate  # On Windows use `venv\Scripts\Activate.ps1`
```

5. Install the required dependencies.

```bash
pip install -r requirements.txt
```

6. Open the Jupyter Notebook file `model.ipynb` in your preferred Jupyter Notebook reader. And run all the cells.
